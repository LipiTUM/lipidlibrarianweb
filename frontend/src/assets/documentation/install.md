## Installation

LipidLibrarian is available as a **Python package**, a **command-line interface**, and a **web application**. Full installation instructions, requirements, and configuration options are maintained in the official repository.

### Python Package

The LipidLibrarian Python package is available on [PyPI](https://pypi.org/project/lipidlibrarian):

```bash
pip install lipidlibrarian
```

### Web Application

The LipidLibrarian Web Application is hosted in its own repository:

- [lipitum/lipidlibrarianweb](https://github.com/lipitum/lipidlibrarianweb)

### Full Documentation & Source Code

For detailed installation guides, dependency information, Docker/container setup, and contribution guidelines, please refer to the official GitHub repository:

- [lipitum/lipidlibrarian](https://github.com/lipitum/lipidlibrarian)
